모아키

모아키는 iPhone을 위한 제스처 한글 키보드입니다. 자음 키를 누른 뒤 방향을 그리면 모음이 입력되고, 한글이 자연스럽게 조합됩니다. 복잡한 전환 없이 iPhone에서 한 손으로 빠르게 입력하는 데 초점을 맞췄습니다.

모아키의 핵심 기능

모아키는 iPhone용 커스텀 키보드 확장과 동반 앱으로 구성되어 있습니다. 앱에서는 설치 방법과 튜토리얼을, 키보드에서는 실제 입력 경험을 제공합니다.

제스처 모음 입력

자음 키 위를 스와이프해서 21개 모음을 입력할 수 있고, 복합 모음과 왕복 제스처도 지원합니다.

간결한 배열

4줄 자음 배열, 길게 누르기 숫자 입력, 별도 기호 키패드로 작은 면적 안에서 빠르게 입력할 수 있습니다.

튜토리얼 제공

동반 앱 안에 활성화 안내와 연습 화면이 포함되어 있어서 처음 사용하는 사람도 흐름을 바로 익힐 수 있습니다.

모아키 활성화 방법

앱 설치 후 iOS 설정에서 키보드를 추가하고, 입력 중 `🌐` 버튼으로 모아키로 전환하면 됩니다.

1

설정 앱을 엽니다.

2

일반 > 키보드 > 키보드로 이동합니다.

3

새 키보드 추가...를 누르고 모아키를 선택합니다.

4

텍스트 입력 중 🌐 버튼을 눌러 모아키로 전환합니다.
